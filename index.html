PixBee

Integre cobranças Pix em qualquer sistema com um simples arquivo .json

O que é o PixBee?

PixBee é um executável (.exe) que permite gerar cobranças Pix dinâmicas com diferentes PSPs, de forma simples e sem precisar lidar com APIs complicadas. Ideal para sistemas legados (Delphi, VB6, etc.) ou integrações rápidas via linha de comando.

Como funciona?

  1. Você gera um arquivo .json com os dados da cobrança
  2. Executa o comando: pixbee.exe parametros.json
  3. Recebe o Pix pronto para uso: chave copia-e-cola, QR code e status
